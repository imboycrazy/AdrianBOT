# AdrianBOT

A Discord bot built with discord.js v14 and TypeScript. For personal use.

## Commands

- `@AdrianBOT` — mention the bot for a random response, 51 avaiable
- `/adrian` — slash command for a random response
- `/askadrian <question>` — ask AdrianBOT a question and get an answer
- `/dinnertime` — ask Adrian what's for dinner
